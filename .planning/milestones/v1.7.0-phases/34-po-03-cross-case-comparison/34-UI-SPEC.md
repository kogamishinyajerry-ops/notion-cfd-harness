## Component Inventory

### ComparisonPage.tsx (NEW)
- Route: `/comparisons/new` or embedded in sweep detail
- Two-panel layout: left (case selector, 320px fixed width), right (ComparisonView with 3 tabs)
- Case selector panel: scrollable list grouped by sweep_id, each group collapsible
- Selected count badge updates live

### CaseSelectorItem
- Checkbox + case hash (monospace, 12px) + param summary (12px, secondary color) + provenance chip row
- States: unchecked, checked (left border accent), hover (bg-secondary), provenance-warning (amber border + warning chip)
- Provenance chip: 4 items inline (OpenFOAM version, mesh hash, solver hash, compiler version) — truncated with ellipsis if > 120px

### ProvenanceWarningBanner
- Full-width banner between selector and ComparisonView
- Amber background (#f59e0b10), amber left border (4px), warning icon
- Dismissible (X button), collapsible
- Shows mismatched fields list

### ComparisonView (3-tab layout)
- Tab bar: horizontal, underline indicator (accent color), 48px height
- Active tab: accent underline 2px, heading weight 600
- Inactive tab: secondary text color
- Tab content: fills remaining height

### ConvergenceOverlay (ConvergenceOverlay.tsx, NEW)
- Recharts LineChart with log-scale YAxis
- Multiple Line entries (one per selected case)
- Series colors from SERIES_COLORS array (cycled if > 8 cases)
- Merged iteration-keyed data: single X-axis (iteration), Y-axis (residual log-scale)
- isAnimationActive={false}
- ResponsiveContainer height: 400px
- Legend at bottom, CustomTooltip for scientific notation

### DeltaFieldViewer (DeltaFieldViewer.tsx, NEW)
- iframe (full tab content area, no CFDViewerBridge — delta viewer is a standalone trame session)
- Launched via API call: POST /comparisons/{id}/delta-session → returns { trame_url }
- Loading state: spinner centered with "Computing delta field..."
- Error state: error message + "Retry" button
- Mesh mismatch warning shown above viewer if provenance.mesh_seed_hash differs

### MetricsTable (MetricsTable.tsx, NEW)
- HTML table with sortable columns
- Columns: Case ID | Params | Final Residual | Execution Time | Diff % (vs reference)
- Sort: click column header → toggle asc/desc, indicator arrow
- Diff % cell: highlight if |diff| > 10% (amber background)
- Export CSV button: top-right, primary button style
- Empty state: "No metrics available"

### ComparisonResult JSON Download
- "Download JSON" button in ComparisonView header
- Uses URL.createObjectURL(Blob) pattern (same as SweepDetailPage CSV export)
- Filename: `comparison_{id}_{timestamp}.json`
