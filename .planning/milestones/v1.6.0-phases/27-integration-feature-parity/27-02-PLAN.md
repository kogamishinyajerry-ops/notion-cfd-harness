## Key interfaces

From CFDViewerBridge.ts:
```typescript
export default class CFDViewerBridge {
  constructor(iframe: HTMLIFrameElement)
  send(msg: BridgeMessage): void   // posts to iframe.contentWindow
  onMessage(handler): () => void    // returns unsubscribe
  destroy(): void                   // removes window listener
}
type BridgeMessage =
  | {type:'field'; field: string}
  | {type:'slice'; axis: 'X'|'Y'|'Z'; origin: [number,number,number]}
  | {type:'slice_off'}
  | {type:'color_preset'; preset: 'Viridis'|'BlueRed'|'Grayscale'}
  | {type:'scalar_range'; mode: 'auto'|'manual'; min?: number; max?: number}
  | {type:'volume_toggle'; enabled: boolean}
  | {type:'timestep'; index: number}
  | {type:'clip_create'; insideOut: boolean; scalarValue: number}
  | {type:'contour_create'; isovalues: number[]}
  | {type:'streamtracer_create'; direction: 'FORWARD'|'BACKWARD'; maxSteps: number}
  | {type:'filter_delete'; filterId: string}
  | {type:'filter_list'}
  | {type:'screenshot'; width: number; height: number}
  | {type:'volume_status'};

type BridgeInboundMessage =
  | {type:'ready'}
  | {type:'fields'; fields: string[]}
  | {type:'volume_status'; enabled: bool; field_name: string|null; gpu_available: bool; gpu_vendor: string; cell_count: number; cell_count_warning: bool}
  | {type:'filter_response'; success: bool; filterId?: string}
  | {type:'filter_list'; filters: Array<{id:string; type:string; parameters:Record}>}
  | {type:'screenshot_data'; image: string}
  | {type:'camera'; position: [n,n,n]; focalPoint: [n,n,n]};
```

From TrameViewer.tsx:
- State machine: 'idle' -> 'launching' -> 'connected' -> 'error' -> 'disconnected'
- ViewerState type: 'idle' | 'launching' | 'connected' | 'disconnected' | 'error'
- Props: jobId: string, caseDir: string, onError?: fn, onConnected?: fn
- Bridge created in useEffect when iframeRef.current is available
- handleBridgeMessage processes: ready, fields, volume_status, filter_response, filter_list, screenshot_data, camera
- All 15 control handlers call bridgeRef.current?.send(...)
- Screenshot: handleScreenshot checks screenshotCapturing guard (debounce), sends {type:'screenshot', width, height}
- Screenshot download: atob(base64Data) -> ArrayBuffer -> Blob -> URL.createObjectURL -> <a download> click -> URL.revokeObjectURL; debounced reset with 500ms setTimeout
- AdvancedFilterPanel wired via bridgeSend() wrapper mapping legacy method names to bridge message types
