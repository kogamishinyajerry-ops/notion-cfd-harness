  <files>dashboard/src/services/CFDViewerBridge.ts</files>
  <action>
Create `dashboard/src/services/CFDViewerBridge.ts` — a framework-agnostic postMessage bridge.

The bridge is a plain TypeScript class with no React dependencies, exposing:
- `constructor(iframe: HTMLIFrameElement)` — stores iframe ref
- `send(msg: BridgeMessage): void` — posts to iframe
- `onMessage(handler: (msg: BridgeInboundMessage) => void): () => void` — returns unsubscribe
- `destroy(): void` — removes listener

The `BridgeMessage` union type covers all outbound commands. The `BridgeInboundMessage` union type covers all inbound events.

Message types (exact field names):

```typescript
// Outbound
type BridgeMessage =
  | { type: 'field';            field: string }
  | { type: 'slice';            axis: 'X' | 'Y' | 'Z'; origin: [number, number, number] }
  | { type: 'slice_off' }
  | { type: 'color_preset';     preset: 'Viridis' | 'BlueRed' | 'Grayscale' }
  | { type: 'scalar_range';     mode: 'auto' | 'manual'; min?: number; max?: number }
  | { type: 'volume_toggle';    enabled: boolean }
  | { type: 'timestep';         index: number }
  | { type: 'clip_create';      insideOut: boolean; scalarValue: number }
  | { type: 'contour_create';   isovalues: number[] }
  | { type: 'streamtracer_create'; direction: 'FORWARD' | 'BACKWARD'; maxSteps: number }
  | { type: 'filter_delete';    filterId: string }
  | { type: 'filter_list' }
  | { type: 'screenshot';       width: number; height: number }
  | { type: 'volume_status' };

// Inbound
type BridgeInboundMessage =
  | { type: 'ready' }
  | { type: 'fields';          fields: string[] }
  | { type: 'volume_status';   enabled: boolean; field_name: string | null;
      gpu_available: boolean; gpu_vendor: string;
      cell_count: number; cell_count_warning: boolean }
  | { type: 'filter_response'; success: boolean; filterId?: string; params?: Record<string, unknown> }
  | { type: 'filter_list';    filters: Array<{ id: string; type: string; parameters: Record<string, unknown> }> }
  | { type: 'screenshot_data'; image: string }
  | { type: 'camera';         position: [number,number,number]; focalPoint: [number,number,number] };
```

Implementation: use `window.addEventListener('message', handler)` scoped to the instance (store bound function ref, remove on destroy). Origin check: `event.origin === 'null'` (local iframe). Call `iframe.contentWindow?.postMessage(msg, '*')`.

Export: `export type { BridgeMessage, BridgeInboundMessage };`
Export the class as default.

**handleLaunch**:
```typescript
const handleLaunch = async () => {
  setViewerState('launching');
  setErrorMessage('');
  try {
    const result = await launchVisualizationSession(jobId, caseDir);
    // session_url is http://localhost:{port}
    // iframe src will be set to result.session_url
    setViewerState('idle'); // iframe not yet loaded
    setSessionUrl(result.session_url);
    setSessionId(result.session_id);
    // Start heartbeat
    startHeartbeat(result.session_id);
  } catch (err) {
    setErrorMessage(err instanceof Error ? err.message : 'Launch failed');
    setViewerState('error');
  }
};
```

**AdvancedFilterPanel wiring** — since AdvancedFilterPanel calls `sendProtocolMessage(createClipFilterMessage(...))`, we need to adapt it. The simplest approach: create a `bridgeSend` wrapper that maps the legacy protocol objects to bridge messages. Pass `bridgeSend` as `sendProtocolMessage` prop.

```typescript
const bridgeSend = useCallback((msg: object) => {
  // Map legacy protocol message types to bridge types
  const m = msg as Record<string, unknown>;
  const method = m.method as string;
  const params = m.params as Record<string, unknown>;
  switch (method) {
    case 'visualization.filters.clip.create':
      bridge?.send({ type: 'clip_create', insideOut: params.insideOut as boolean, scalarValue: params.scalarValue as number });
      break;
    case 'visualization.filters.contour.create':
      bridge?.send({ type: 'contour_create', isovalues: params.isovalues as number[] });
      break;
    case 'visualization.filters.streamtracer.create':
      bridge?.send({ type: 'streamtracer_create', direction: params.integrationDirection as 'FORWARD'|'BACKWARD', maxSteps: params.maxSteps as number });
      break;
    case 'visualization.filters.delete':
      bridge?.send({ type: 'filter_delete', filterId: params.filterId as string });
      break;
    case 'visualization.filters.list':
      bridge?.send({ type: 'filter_list' });
      break;
    // screenshot handled via viewport dimensions
  }
}, [bridge]);
```
