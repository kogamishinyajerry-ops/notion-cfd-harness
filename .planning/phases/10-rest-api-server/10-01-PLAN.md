# Phase 10 Plan 01 — REST API Core

## Metadata

- **Plan ID:** 10-01
- **Phase:** 10 — REST API Server
- **Milestone:** v1.2.0
- **Status:** Planning
- **Gap Closure:** false

## Goal

Create FastAPI-based REST API with core endpoints for case management, job submission, and status monitoring.

## Context

After Phase 9 (Report Automation), the system has comprehensive CLI capabilities. Phase 10 exposes these via REST API for external integrations and programmatic access.

## Tasks

| Task | Description |
|------|-------------|
| 10-01-01 | Set up FastAPI project structure with Uvicorn |
| 10-01-02 | Create API endpoints for case CRUD operations |
| 10-01-03 | Create job submission and status endpoints |
| 10-01-04 | Add OpenAPI/Swagger documentation |
| 10-01-05 | Write unit tests for API endpoints |

## Deliverables

- `api_server/` directory with FastAPI application
- REST endpoints: `/cases`, `/jobs`, `/status`
- OpenAPI documentation at `/docs`
- Unit tests in `tests/test_api_*.py`

## Dependencies

- Phase 9 (Report Generator) must be complete

## Notes

- Use existing CLI modules (cli.py, runtime.py) as backend
- Maintain PermissionLevel checks from Phase 3
- Support both synchronous and async job submission
