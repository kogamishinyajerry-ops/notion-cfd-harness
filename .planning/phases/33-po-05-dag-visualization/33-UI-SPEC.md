## Node Detail Panel

The node detail panel opens as a **side drawer** (not a modal overlay) on the right side of the DAG canvas.

| Field | Source |
|-------|--------|
| Step ID | step.id |
| Step type | step.step_type |
| Status | step.status (colored badge) |
| Duration | formatDuration(step.started_at, step.completed_at) |
| Depends on | step.depends_on (joined list, or "none") |
| Parameters | JSON.stringify(step.params, null, 2) in monospace block |
| Result summary | step.result.status, step.result.exit_code |
| Diagnostics | step.result.diagnostics (if present) |

Drawer width: 360px. Background: var(--bg-primary). Close on X click or clicking the DAG background.

## WebSocket Integration

The existing `pipelineWs` service is reused without modification.

| WebSocket Event | DAG Reaction |
|-----------------|---------------|
| step_started | Find node by step_id, update background to RUNNING color, clear warning icon |
| step_completed | Update node to COMPLETED green |
| step_failed | Update node to FAILED red |
| pipeline_completed | All RUNNING nodes transition to terminal states |
| pipeline_failed | All RUNNING nodes transition to FAILED |
| pipeline_cancelled | All RUNNING nodes transition to CANCELLED (gray) |

**Warning icon rule:** A RUNNING step shows a warning icon if `Date.now() - new Date(step.started_at).getTime() > expected_duration_ms`. The `expected_duration_ms` per step_type is:

| step_type | Expected Duration |
|-----------|-------------------|
| generate | 60000 (1 min) |
| run | 300000 (5 min) |
| monitor | 600000 (10 min) |
| visualize | 120000 (2 min) |
| report | 60000 (1 min) |

## Component Inventory

### DAGCanvas (new component)
- Wraps `<ReactFlow>` with custom node types
- Props: `steps: PipelineStep[]`, `onNodeClick: (step: PipelineStep) => void`
- States: idle, loading, empty (no steps)

### PipelineStepNode (custom React Flow node)
- Custom node component registered with `registerNode`
- States: PENDING (gray bg), RUNNING (blue bg + animated border), COMPLETED (green bg), FAILED (red bg), SKIPPED (yellow bg)
- Warning badge: amber triangle icon, top-right corner, visible when RUNNING > expected duration
- Shows: step_id (bold), step_type badge, status text, duration

### NodeDetailDrawer (new component)
- Side panel: 360px fixed width, slides in from right
- Props: `step: PipelineStep | null`, `onClose: () => void`
- Sections: header with step_id + close button, params block, result block, diagnostics block

### DAGControls (uses @xyflow/react Controls component)
- Position: bottom-right of canvas
- Buttons: zoom-in, zoom-out, fit-view

### DAGMinimap (uses @xyflow/react MiniMap component)
- Position: bottom-right above controls
- Shows: all nodes as colored rectangles
