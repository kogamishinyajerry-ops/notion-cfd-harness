**Interface:**
```typescript
interface ResidualChartProps {
  jobId: string;
  onClose?: () => void;
}
```

**State:**
- `residualData: Array<{iteration: number; time: number; Ux?: number; Uy?: number; Uz?: number; p?: number}>` - max 500 points
- `isConnected: boolean`

**useResidualStream hook logic:**
1. Connect to WebSocket on mount via `wsService.connect(jobId)`
2. Subscribe via `wsService.subscribe(jobId, (message) => {...})`
3. On message.type === 'residual':
   - Parse message as ResidualMessage
   - Append to residualData: { iteration, time_value, ...residuals }
   - Maintain 500-point sliding window: if data.length > 500, shift oldest
4. Cleanup: unsubscribe and disconnect on unmount

**Chart implementation:**
- Use Recharts ResponsiveContainer (width="100%", height={400})
- XAxis: dataKey="iteration", label="Iteration"
- YAxis: type="logDomain", domain={[1e-8, 1e-1]}, tickFormatter for scientific notation
- CartesianGrid with logScale=true
- Four Line components with dataKey for Ux (red), Uy (green), Uz (blue), p (purple)
- Tooltip showing iteration, time, and all residual values
- Legend showing color-to-variable mapping
- Update trigger: data={residualData} always animated=false for performance

**CSS (ResidualChart.css):**
- Container with border, border-radius, padding
- Header with "Live Convergence Chart" title and close button
- Chart area with proper sizing
- Connection status indicator (green dot when connected)
